# IBM i DevOps Migration Lab

This project simulates migrating legacy IBM i source code into a modern DevOps workflow using Git and CI/CD pipelines.

## Features

- Legacy code dependency analysis
- Source validation
- Artifact creation
- CI/CD pipeline simulation

## Pipeline Stages

1. Dependency Analysis
2. Validation
3. Build
4. Deploy

## Tools Used

- Python
- Bash
- Git
- Azure DevOps Pipeline (YAML)# ibmi-devops-migration
